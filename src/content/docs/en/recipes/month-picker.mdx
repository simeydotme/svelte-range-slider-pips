Here's an example of a nice looking month picker slider with labels inside the handles.
There's two key parts to this example;
- Set the `formatter` function to format the values as a month name
- Use the `css` to move the `.rangeFloat` label on top of the `.rangeHandle`

Of course, the normal options to enable `floats` and the `range` are required.

### Arbitary Labels

The `formatter` function can be used to format the values as any arbitrary string.
See the [Formatter example](../examples/formatter#multiple-formatters) for more information.
