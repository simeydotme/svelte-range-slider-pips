Here is a plausible use case, showing a slider to choose a price range for hotels and
showing the number of hotels for each price in the range, plus the total number of hotels.

<Demo name="Hotel Range" panel="svelte" client:load>
  <HotelRange client:only="svelte" />
</Demo>

First of all we are bucketing the hotels into price ranges (e.g. $0-$10, $10-$20, etc.), and
then we are creating a `<RangeSlider>` with a `step` that matches the bucket size.

We are using the `formatter()` prop to add a special label for each pip that sets a `--h` custom property
to match the number of hotels in each bucket. Then CSS is applied to style the height of the label to match the number of hotels.
