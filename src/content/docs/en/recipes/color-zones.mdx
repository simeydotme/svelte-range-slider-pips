There may be scenarios where you want to show a part of the Range Slider to be
either safe, unsafe or perhaps both! 

You could imagine creating extra `<div/>` elements which color a certain portion
of the range to highlight a certain area. But this is essentially a 'style', and so
the correct way to do this should be using `stylesheets` (aka; CSS).
