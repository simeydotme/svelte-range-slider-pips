By passing the float prop to the component, we can have a nice label which floats 
above the handle and shows the current value.

<Info>Hover/Drag the Slider handles below to see the float</Info>

<Demo name="Floating Labels" client:visible>
  <Float client:visible />
  <div slot="svelte">
    ```svelte
    <RangeSlider float />
    ```
  </div>
</Demo>

<Note type="info">
  I manually set the above Slider to be focussed on load so that the float is
  visible. [See below for how to do this](#onload).
</Note>






## Float without Hover

The Slider can be initialised with `hoverable` set `false` which will prevent
the float from appearing on hover.

<Demo name="Floating Labels" client:visible>
  <NoHover client:visible />
  <div slot="svelte">
    ```svelte
    <RangeSlider float />
    <RangeSlider float hoverable={false} />
    ```
  </div>
</Demo>

<Note type="warning">
  This will also **prevent all other hover effects** on the Slider. If you wish
  only to hide the hover effect for floats, you may edit the CSS for more control.
</Note>



## Always show the floats

We can set the floats to always be visible by modifying the css slightly.

<Demo name="Always Shown Floats" client:visible>
  <Always client:visible />
  <Code slot="svelte" code={cleanImport(AlwaysRaw)} lang="svelte" />
</Demo>

<Note type="info">
  I manually set the above Slider to be focussed on load so that the float is
  visible.
</Note>
