The sliders come, by default with a Svelte Spring animation. This uses the internal
`spring()` function from Svelte. The properties of the `spring()` (`stiffness`, `damping`)
can be set with this `springValues` property.


## Default Spring

The default animation is **not especially noticeable when moving the handle** manually. But when
clicking from one end to the other, the **large distance travelled is animated** and much
more noticeable.

<Demo name="The default spring" client:visible>
  <DefaultEasing client:visible />
  <Code slot="svelte" code={cleanImport(DefaultEasingRaw)} lang="svelte" />
</Demo>



## Disable Spring

By default, the spring is enabled. To disable it, set the `spring` property to `false`.
This will disable animations and make the handles move instantly to the new position.

<Demo name="Disable Spring" client:visible>
  <DisabledEasing client:visible />
  <Code slot="svelte" code={cleanImport(DisabledEasingRaw)} lang="svelte" />
</Demo>



## Custom Spring
