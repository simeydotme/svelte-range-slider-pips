The sliders come, by default with a Svelte Spring animation. This uses the internal
`spring()` function from Svelte. The properties of the `spring()` (`stiffness`, `damping`)
can be set with this `springValues` property.


## Default Spring

The default animation is **not especially noticeable when moving the handle** manually. But when
clicking from one end to the other, the **large distance travelled is animated** and much
more noticeable.
