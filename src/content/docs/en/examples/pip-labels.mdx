Each pip has an intrinsic value based on the `min` and `max` and its position
in the Slider. This value can be displayed as a `label` over the `pip`.

The labels are displayed using the `all`, `first` `last` and `rest` props.

<Note title="Demo" type="info">
  [Try the pips/labels configuration demo](./pips#control-the-display)
</Note>
