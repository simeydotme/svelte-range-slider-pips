Here we demonstrate the **Slider's main feature; "notches"**, or as I call them; **"`pips`"**. These little markers sit below 
the Slider by default to mark regular intervals along the Slider's length.

</Note>




## Control the Display

Pip display is controlled by the properties; `all`, `first`, `last` and `rest`. 
These properties can be set to `true` or `false` to control the display of the pips.

This is also true for `labels`. [See more here](./pip-labels)
